= Hilfe für das Plugin "IMA Alarm Fernüberwachung"

== Beschreibung
Dieses Plugin dient zur Anzeige des Zustandes Ihrer IMA Alarm Fernüberwachung (wird insbesondere vom MACIF vermarktet).


== Voraussetzung
Sie müssen zuerst die Option "Fernsteuerung" Ihres Alarms aktivieren.
Dies kann kostenlos bei IMA Fernüberwachung durchgeführt werden.
Sie erhalten dann ein Login/Passwort um auf die Seite zuzugreifen : https://pilotageadistance.imateleassistance.com .

 
== Alarm erstellen
Sobald das Plugin installiert ist:

- klicken Sie auf Plugins > Sicherheit > IMA Alarm
- klicken Sie dann auf das Symbol '"+"
- in dem sich öffnenden Fenster wählen Sie einen Gerätenamen, zum Beispiel: "Mein Alarm".
- Sie gelangen auf die Konfigurationsseite Ihres Alarms:

[.text-center]
image::doc.png[Konfiguration]

[NOTE]
===
Mit dem hier zu konfigurierenden Login/Passwort können Sie auf https://pilotageadistance.imateleassistance.com zugreifen (siehe Abschnitt Voraussetzungen).
Diese IDs sind nur auf Ihrem Jeedom gespeichert und werden verwendet, um den Zustand des Alarms abzurufen. 
===

- klicken Sie abschließend auf Speichern.

[IMPORTANT]
====
Denken Sie daran

*  ein Elternobjekt zu wählen,
*  auf aktivieren zu klicken,
* auf sichtbar zu klicken.

Ohne diese 3 Bedingungen sehen Sie Ihren Alarm nicht auf dem Jeedom-Armaturenbrett.
====


== Den Zustand des Alarms ansehen
Klicken Sie nun auf Start > Armaturenbrett : Ein neues Widget wird angezeigt, das den aktuellen Zustand des Alarms anzeigt.

[.text-center]
image::widget.png[Widget]

3 zustände sind möglich:

- AN (der Alarm ist eingeschaltet), 
- AUS (der Alarm ist ausgeschalten), 
- PARTIELL (der Alarm ist nur bei einem Teil Ihres Hauses aktiv).
